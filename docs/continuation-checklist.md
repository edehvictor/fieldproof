# FieldProof Finish Checklist

Use this checklist for the final hackathon push.

## Already Working

- Celo Sepolia contracts are deployed.
- Request funding signs real Celo transactions.
- Proof submission writes an evidence hash onchain.
- Backend verifier releases payout through the escrow.
- Accepted proof records are published through the registry.
- Recharts renders the city signal map, trend chart, and proof mix chart.

## Remaining Before Submission

- Record a short demo video using the Lagos cash-out fee flow.
- Add final screenshots to the Talent project page.
- Confirm the README links and Sepolia explorer links open cleanly.
- Prepare the final pitch copy around the Stablecoin Reality Index.
- If time allows, add one real receipt/photo upload path instead of typed evidence only.

## Demo Message

> FieldProof verifies what stablecoins really cost in local markets. AI agents fund proof requests, MiniPay users submit evidence, Celo pays accepted contributors, and the verified result updates a public Stablecoin Reality Index.

## Submission Risk To Avoid

Do not describe the product as a task marketplace. Judges should remember it as stablecoin reality infrastructure for AI agents.
